## Git

### Init

#### Init with code

{% highlight bash %}
git init
git remote add ...
{% endhighlight %}

#### Init by clone the repo

{% highlight bash %}
git clone ...
{% endhighlight %}

### Daily push

<pre>
<code class="bash">
git pull
git status
git diff
git add .
git add --all
git commit -asm "comment"
git push
</code>
</pre>

### Branching

<pre>
<code class="bash">
git branch -b newbranch
git add new.txt
git commit -asm "comment"
git checkout master
git merge newbranch
</code>
</pre>

### Submodule

<pre>
<code class="bash">
git submodule add git://github.com/quxiaofeng/mycv.git _cv
git add .
git commit -asm "added a submodule"
git push
</code>
</pre>
